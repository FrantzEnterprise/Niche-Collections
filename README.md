# Niche Collections

A collection of single-file, zero-infrastructure service intake apps for trade businesses. Each app is a self-contained HTML file that lets customers describe what they need and send it as a pre-written text or email - no app stores, no monthly fees, no data collection.

## Available

- [LockCMD](https://frantzenterprise.github.io/LockCMD/) - Locksmith service intake
